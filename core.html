Core IRC Functions¶

Bot Events¶

The event system allows you to watch for an event, and register a function to be called. Examples of events would be some of the IRC message types such as PRIVMSG, JOIN, PART, QUIT, etc.

irc.event(event_name, param=None)¶

Run events on named queue :param event_name: Examples PRIVMSG, CONNECT, JOIN :type event_name: str :param param: Parameters to send to the registered functions. Varies from event to event

irc.event_register(event_name, function)¶

Register a new event :param event_name: Examples PRIVMSG, CONNECT, JOIN :type event_name: str :param function: function to be called. Order is not guarenteed :type function: func

irc.event_unregister(event_name, function)¶

Unregister an event

Parameters:
  • event_name (str) – Examples PRIVMSG, CONNECT, JOIN
  • function (func) – function to be unregistered
